# ALIVE NEWS — Modern Editorial News Platform

A modern editorial-style news platform inspired by WIRED, built with Next.js App Router, TypeScript, and Tailwind CSS.

Designed and developed by **Ketan**.

---

## 🚀 Overview

ALIVE NEWS is a premium magazine-style news application that delivers real-time news using the NewsAPI service. The project focuses on modern UI/UX, scalable architecture, responsive layouts, and production-level development practices.

This project is built using the latest Next.js architecture with App Router and Server Components for improved SEO, performance, and scalability.

---

## ✨ Features

- **Modern WIRED-Inspired Editorial UI**: High-contrast, publication-grade dark design interface.
- **Responsive Fluid Layouts**: Fully optimized across mobile, tablet, laptop, and ultra-wide desktop monitors.
- **Real-Time News Integration**: Native connectivity with live media endpoints.
- **Category-Based News Sections**: Structured taxonomy for seamless topic discovery.
- **Featured Hero Articles**: Prominent focal sections optimized for structural content emphasis.
- **Optimized Image Rendering**: Next-generation media layout engine minimizing Layout Shift (CLS).
- **Dynamic Routing**: Clean, semantic, nested URI routing.
- **Server-Side Rendering & ISR**: Performance-first hydration rendering models for rapid page loads.
- **SEO-Friendly Architecture**: Deeply integrated semantic HTML and metadata generation tags.
- **Smooth UI Interactions**: Fluid micro-animations for elevated user satisfaction.

## ⚙️ Installation

### 1. Clone Repository

    ```bash
    git clone [https://github.com/ketangodhani/alive.git](https://github.com/ketangodhani/alive.git)
    cd alive
    ```

### 2. Install Dependencies

    ```bash
    npm install
    ```

### 3. Setup Environment Variables

    Create a `.env` file in the root directory:
    ```env
    NEXT_PUBLIC_NEWS_API_KEY=YOUR_API_KEY
    ```
    💡 Note: Obtain your API credential from the official NewsAPI Portal.

### 4. Start Development Server

    ```bash
    npm run dev
    ```
